## Messenger/Slack clone

An instant chat messenger built with Firebase Cloud.

Full capabilties:

- realtime messaging
- create channels
- direct messages
- likes
- uploadphotos
- user authentication
- typing indication
- online indication
- dark mode

npm i && npm run start or yarn && yarn start
